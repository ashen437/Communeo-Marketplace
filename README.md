<h3 align="left">ğŸ”§ Installation</h3>

###

<p align="left">
  1. Clone the repository<br>
  2. Install dependencies:<br>
</p>

```bash
# For backend
cd api
npm install

# For frontend
cd client
npm install
```

<p align="left">
  3. Set up environment variables in both frontend and backend .env files<br>
  4. Run the development servers:<br>
</p>

```bash
# For backend

cd client
npm run dev
```
